# 常用函数
```
#行向量
x = [1 2 3 4 5]

#列向量
x = [1; 2; 3; 4; 5]

#随机数向量1X3 & 3X3
x = rand(1,3)
x = rand(3,3)

#绘制直方图
hist(x)
hist(x,50)

#绘制曲线图
plot(x,y)

#高斯分布
hist(log(x),50)

#均值
mean(x)

#标准方差
std(x)

```
